# Bulk Scheduler - Monthly Content Planning

Create and schedule an entire month's worth of content in one session.

## Overview

The Bulk Scheduler helps you batch create and schedule videos in advance, maintaining consistency without daily posting.

## Two Ways to Access

### Method 1: Direct from Dashboard
- Click "Bulk Scheduler" button on main dashboard
- Access setup options immediately

### Method 2: From Faceless Video Generator
- When creating a video, reach Step 2 (Schedule Setup)
- Toggle on "Bulk Schedule" option
- Guided through same workflow

## Complete Workflow

### Step 1: Setup Options

**Number of Videos**:
- Choose how many videos to create
- Typically 30-60 for a month

**Start Date**:
- When to begin posting
- Can start immediately or future date

**Schedule Duration**:
- Up to 30 days maximum
- Set end date

**Posting Frequency**:
- 1 post per day
- 2 posts per day (recommended: 9 AM, 7 PM)
- 3+ posts per day (aggressive)
- Custom schedule

### Step 2: Video Configuration

**Video Style**:
- Choose preferred style (AI Visuals, B-Roll, etc.)
- Applied to all videos

**Voice & Language**:
- Select voice actor
- Choose language
- Consistent across all videos

**Other Settings**:
- Subtitle style
- Video length
- Aspect ratio
- All standard video options

### Step 3: Script Generation

**Enter Topic**:
- Main topic or niche
- Can be broad (e.g., "AI for business")
- AI creates variations

**AI Script Writer**:
- Automatically creates unique scripts
- Each video gets different script
- All based on your original topic
- Varied but related content

### Step 4: Generate Videos

**Review Summary**:
- Total number of videos
- Credits required
- Schedule overview

**Generate**:
- Click "Generate Videos"
- System creates all videos
- Redirected to Bulk Scheduler Tracker

### Step 5: Monitor Progress

**Bulk Scheduler Tracker**:
- Dashboard to monitor all scheduled content
- Track each video's status
- See what's pending, processing, or published
- Adjust schedules if needed

### Step 6: Automatic Publishing

**Once Generated**:
- Syllaby publishes automatically
- Based on channels, timing, frequency
- No manual intervention needed
- Content goes live on schedule

## Best Practices

### Planning

1. **Plan at Month Start**: Set aside 2 hours monthly
2. **Choose Optimal Times**: 9 AM and 7 PM work best
3. **Mix Content Types**: Variety keeps audience engaged
4. **Review Topics**: Ensure good mix of content

### Content Strategy

1. **Consistent Style**: Use same style for brand recognition
2. **Varied Topics**: Different angles on same niche
3. **Series Content**: Create related video series
4. **Evergreen + Trending**: Mix both types

### Optimization

1. **Monitor Tracker**: Check progress regularly
2. **Adjust if Needed**: Can modify schedules
3. **Track Performance**: See what works
4. **Iterate**: Improve next month's batch

## Monthly Planning Session

### Sunday: Planning (30 minutes)

1. Review last month's performance
2. Identify top-performing content
3. Plan topics for new month
4. Prepare for bulk creation

### Monday: Creation (2 hours)

1. Open Bulk Scheduler
2. Configure all settings
3. Generate scripts
4. Create all videos
5. Review and approve

### Rest of Month: Maintenance

1. Monitor publishing
2. Engage with comments
3. Track performance
4. Create additional content if needed

## Pro Tips

### Efficiency

1. **Batch Everything**: Create all content at once

### Quality

4. **Iterate**: Improve each month

## Common Questions

**Q: How many videos should I create?**  
A: 30-60 videos for a month (1-2 per day)

**Q: How long does it take?**  
A: 2 hours for 60 videos (setup + generation)

**Q: Can I edit after bulk creation?**  
A: Yes! Edit individual videos as needed

**Q: What if I need to change schedule?**  
A: Use Bulk Scheduler Tracker to adjust

**Q: Can I mix different styles?**  
A: Bulk scheduler uses one style, but you can create separate batches
