# Dashboard Overview

Understanding your Syllaby dashboard is key to efficient content creation.

## Main Dashboard Elements

### 1. Suggested Video Topic (Front and Center)
- AI-generated topic suggestions based on your activity
- Click to start creating immediately
- Updates based on your niche and preferences

### 2. Favored Topics
- Saved ideas from past suggestions
- Revisit anytime without searching
- Build your content library

### 3. Quick Action Buttons
- **Bulk Creation**: Batch create multiple videos
- **Single Video Creation**: Create one video at a time
- Fast access to most common actions

### 4. Faceless Video Samples
- Preview different AI visual styles
- See exactly what to expect before creating
- Test different styles for your niche

### 5. Left Sidebar Tools
- **Idea Discovery**: Find trending topics
- **Faceless Video Generator**: Create videos
- **Bulk Scheduler**: Plan content in advance
- **Content Calendar**: Manage your schedule
- **Analytics**: Track performance
- **Thumbnail Generator**: Create thumbnails
- **Real Clone**: Create your avatar
- **Settings**: Account configuration

### 6. Top Navigation
- **Quick "Create Video" Button**: One-click video creation
- **Credit Balance**: See remaining credits
- **Credit Usage History**: Track your usage
- **Account Settings**: Email, subscription, storage

## Key Areas to Know

### My Videos Section
- View all created videos
- Edit existing videos
- Download videos
- Track publishing status

### Assets Library
- Previously generated images
- Uploaded files
- Stock clips used
- Reusable content (free to use)

### Content Calendar
- Visual schedule of all videos
- See what's publishing when
- Adjust schedules easily
- Multi-platform view

## Pro Tips

1. **Check Suggested Topics Daily**: Fresh ideas appear regularly
2. **Save Interesting Topics**: Add to Favored Topics for later
3. **Monitor Credit Usage**: Plan your content creation
4. **Use Quick Create**: Fastest way to start a video
5. **Explore Samples**: Test styles before committing credits
