# Daily Content Workflow (5 Minutes)

Create professional faceless videos in just 5 minutes using this proven daily workflow.

## The 5-Minute Workflow

### Step 1: Choose Topic (1 minute)

**Option A: Use Suggested Topics**
- Check dashboard for suggested topics
- Click to start creating
- AI-generated based on your activity

**Option B: Use Idea Discovery**
- Go to Idea Discovery
- Enter broad topic
- Get 10 trending ideas
- Pick one with high search volume

**Option C: Your Own Topic**
- Enter topic directly
- Use AI Script Assistant
- Generate script automatically

### Step 2: Generate Script (30 seconds)

1. Click "AI Script Assistant"
2. Enter your topic
3. Click "Generate Script"
4. Review (edit if needed)

**Pro Tip**: Let AI write the script - it's trained on proven formulas.

### Step 3: Configure Video (1 minute)

**Style**: AI Visuals (most versatile)

**Settings**:
- **Portrait** for TikTok/Reels/Shorts
- **Landscape** for YouTube
- **Under 1 minute** for shorts
- **5-10 minutes** for long-form

**Voice**: Choose your preferred voice actor

**Language**: Your target language

### Step 4: Schedule Publishing (1 minute)

**Platforms**:
- Select all connected platforms
- Or choose specific ones

**Options**:
- **Publish Now**: Immediate
- **Schedule**: Choose time
- **Auto-generate**: Description and tags ✓

**CTA**: Add affiliate link or call to action

### Step 5: Generate Video (30 seconds)

1. Review settings
2. Click "Generate Video"
3. Wait 1-2 minutes
4. Done!

## Daily Routine
